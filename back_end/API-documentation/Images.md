## Upload Post Image

-   Use route (POST): /uploadpostimage/{postID}
-   Returns a string indicating "Bad Request" or "File {name} uploaded successfully!"
-   **_NOTE: Requires a postID, which can be retrieved from making a post (refer to Posts.md)_**
